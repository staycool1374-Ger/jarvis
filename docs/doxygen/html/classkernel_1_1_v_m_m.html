Detailed Description

Virtual memory manager for x86-64 4-level paging.

Note
Maps virtual to physical pages; supports user/kernel page flags.

uint64_t kernel::VMM::clone_kernel_pml4 ( )
static

Creates a fresh PML4: zeroes user entries (0-255), copies kernel entries (256-511). Used during exec/load — NOT for fork (which needs parent user entries).

Returns
Physical address of the new PML4, or 0 on failure.
