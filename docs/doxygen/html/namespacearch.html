◆ handle_gic_irq()

void arch::handle_gic_irq ( void )

Main IRQ handler called from vectors.S IRQ entries. Handles both GICv2 (memory-mapped) and GICv3 (system register) IAR/EOIR.

Note
Spurious interrupts (INTID >= 1023) are acknowledged and discarded. Only the timer PPI (INTID 30) is dispatched to the IDT.
