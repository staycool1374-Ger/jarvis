uint64_t kernel::g_user_access_recover_ip

Recovery address for safe user memory access. Non-zero while we are inside a user memory copy attempt. Page fault handler checks this and redirects here on fault.

Global recovery IP for safe user memory access. Non-zero means we are in a user memory access recovery zone. If a fault occurs and this is set, execution is redirected here.
