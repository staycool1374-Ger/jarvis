◆ prd_from_sg()

size_t kernel::dma::prd_from_sg ( PrdTable & prd,
const SgList & sg,
bool eot = true )

Build a PRD table from a scatter-gather list. ATA bus master PRD format: 32-bit phys addr, 16-bit byte count-1, 16-bit flags. The table is suitable for programming a PCI bus master controller.

Parameters
prdOutput PRD table (must have backing memory accessible by device).
sgInput scatter-gather list.
eotIf true, sets the EOT flag on the last entry.
Returns
Number of PRD entries written.
