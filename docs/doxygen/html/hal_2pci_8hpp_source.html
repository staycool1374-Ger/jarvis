21// ---------------------------------------------------------------------------
22// Architecture-specific PCI config space access
23// x86_64: CF8/CFC port I/O (0xCF8 / 0xCFC)
24// aarch64 / riscv64: ECAM memory-mapped (QEMU virt at 0x100000000)
25//
26// All access functions work on dword-aligned addresses internally, matching
27// the x86 CF8/CFC semantics where bits [1:0] of the register offset select
28// the byte/word within the dword. This ensures identical behavior across
29// all architectures.
30// ---------------------------------------------------------------------------
