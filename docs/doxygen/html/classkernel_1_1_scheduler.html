Detailed Description

Preemptive, rate-monotonic scheduler managing up to MAX_TASKS tasks.

Note
Scheduler is tick-driven and supports periodic and aperiodic tasks.

◆ cleanup_zombies()

void kernel::Scheduler::cleanup_zombies ( )
staticnoexcept

Removes all tasks from the scheduler that have invalid magic (freed-and-reused TCBs) without touching the TCB memory. Also removes tasks with valid magic that are not the idle task, the current task, or known daemons.

Iterates at most CONFIG_MAX_TASKS entries in the task array (defensive bound against a corrupted task_count_). Tasks to keep are compacted to the front of the array; task_count_ is updated accordingly.

Parameters
shell_taskPointer to the shell task (use set_shell_task() beforehand).
