Detailed Description

Cross-architecture tests that validate identical behaviour on all three supported architectures (x86_64, aarch64, riscv64).

Each test exercises a kernel subsystem through its generic interface (VMM, ArchContextManager, Timer, ArchInterruptController, IPC, VFS) and is compiled for all arches. Architecture-specific assertions are guarded with #if/#elif/#endif.
